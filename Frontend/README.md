# MediAssist Frontend

React + Vite UI for MediAssist.

## Features

- Symptoms textarea input
- Optional language selection (English default)
- Loading state (`Analyzing...`)
- Error handling for API failures
- Risk color coding:
	- `HIGH` -> red
	- `MEDIUM` -> orange
	- `LOW` -> green
- Medical disclaimer display

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
